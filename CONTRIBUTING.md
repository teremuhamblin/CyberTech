# Contribuer à Assistant‑IA

Merci de votre intérêt pour le projet.  
Les contributions sont les bienvenues : simples, propres et utiles.

## Comment contribuer
- Fork du projet
- Créer une branche dédiée (`feature/nom`, `fix/nom`)
- Faire des commits clairs et courts
- Ouvrir une Pull Request

## Règles de code
- JS modulaire (un fichier = un module)
- CSS organisé (thèmes, animations, composants)
- Pas de code mort
- Pas de duplication inutile

## Issues
- Décrire clairement le problème
- Ajouter étapes de reproduction
- Indiquer la version utilisée

## Pull Requests
- Explication courte
- Changements limités et cohérents
- Tests manuels effectués

Merci de garder le projet propre, lisible et maintenable.
